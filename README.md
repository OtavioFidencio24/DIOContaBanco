# Desafio Conta Banco
Simulador de Banco

## Descrição do Projeto

Desafio proposto na trilha Back-End da DIO para a criação de um simulador de banco,
onde o usuário digita as informações da conta via terminal.


## Tecnologia Utilizada

- Java 21+
